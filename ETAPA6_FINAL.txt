1️⃣  ANÁLISE DO PROBLEMA
──────────────────────────────────────────────────────────────────────────────

ARQUIVO ANALISADO: src/services/fontService.js

PROBLEMA IDENTIFICADO:
  • A função carregarFontes(pdfDoc) era chamada em CADA requisição
  • Dentro da função, fs.readFileSync() lia 6 arquivos TTF do disco
  • Isso resultava em ~2.4 MB de I/O por requisição
  • Latência adicional: 10-50ms por PDF gerado

CÓDIGO PROBLEMÁTICO (antes):
```javascript
async function carregarFontes(pdfDoc) {
  const ler = (nome) => {
    try {
      return fs.readFileSync(path.join(process.cwd(), nome));  // ❌ I/O a cada chamada
    } catch {
      return null;
    }
  };
  
  return {
    regular:    await embutir(ler(FONT_PATHS.cormorantRegular)),
    bold:       await embutir(ler(FONT_PATHS.cormorantBold)),
    italic:     await embutir(ler(FONT_PATHS.cormorantItalic)),
    boldItalic: await embutir(ler(FONT_PATHS.cormorantBoldItalic)),
    sans:       await embutir(ler(FONT_PATHS.openSansRegular)),
    sansBold:   await embutir(ler(FONT_PATHS.openSansBold)),
  };
}
```

2️⃣  SOLUÇÃO IMPLEMENTADA
──────────────────────────────────────────────────────────────────────────────

TÉCNICA: Pré-carregamento em memória com IIFE (Immediately Invoked Function)

IMPLEMENTAÇÃO:

a) Cache de Bytes (topo do módulo):
   └─ Implementado IIFE que executa UMA ÚNICA VEZ na inicialização
   └─ Lê todos os 6 arquivos TTF do disco uma única vez
   └─ Armazena bytes em memória no objeto FONT_BYTES_CACHE

b) Reutilização:
   └─ Função carregarFontes() usa FONT_BYTES_CACHE
   └─ Nenhum fs.readFileSync() é executado por requisição
   └─ Bytes já carregados são reutilizados

CÓDIGO OTIMIZADO (depois):
```javascript
// ✅ PRÉ-CARREGADO UMA ÚNICA VEZ
const FONT_BYTES_CACHE = (() => {
  const dir = process.cwd();
  
  const lerFonte = (nome) => {
    try {
      return fs.readFileSync(path.join(dir, nome));  // ✅ Executa UMA VEZ
    } catch {
      return null;
    }
  };

  return {
    cormorantRegular:    lerFonte(FONT_PATHS.cormorantRegular),
    cormorantBold:       lerFonte(FONT_PATHS.cormorantBold),
    cormorantItalic:     lerFonte(FONT_PATHS.cormorantItalic),
    cormorantBoldItalic: lerFonte(FONT_PATHS.cormorantBoldItalic),
    openSansRegular:     lerFonte(FONT_PATHS.openSansRegular),
    openSansBold:        lerFonte(FONT_PATHS.openSansBold),
  };
})();

// ✅ REUTILIZA BYTES EM MEMÓRIA
```
